BoomCMS is a content management system which is designed to be easy for editors and developers alike

Features

  • Simple text editing

    Edit pages in-view with the WYSIWYG text editor

  • Open source

    BoomCMS is released under the MIT license and builds on the great work of other open source projects

  • Modular

    A modular design makes it easy to create themes and packages to enhance your site

Install

BoomCMS can be installed via composer

composer create-project boomcms/boomcms --prefer-dist

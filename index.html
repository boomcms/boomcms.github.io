Simple content management for editors and developers

Features

  • Simple text editing

    Edit pages in-view with the WYSIWYG text editor

  • Advanced asset management

    Enhance your webpages with images, video, and PDFs using the integrated asset manager

  • Role base permissions

    Easily manage your team's access to CMS functionality with the role based permissions system

  • Open source

    BoomCMS is released under the MIT license and builds on the great work of other open source projects

  • Built on Laravel

    Being built on the modular Laravel PHP framework makes it easy to add functionality to Boom or integrate it with other systems

  • Actively maintained

    Regular updates introduce new features and improvements

Install

BoomCMS can be installed via composer

composer create-project boomcms/boomcms
